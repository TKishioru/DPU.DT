>> งานเดี่ยว
>> สามารถใช้ไลบรารี่ Bootstarap หรือไลบรารี่ชนิดอื่นๆ ได้
>> มีหน้าเว็บอย่างน้อย 3 หน้า สามารถลิงก์กลับไปกลับมาได้
>> ใส่ข้อมูลเป็นประวัติส่วนตัว หรือประวัติสิ่งของ-สถานที่ต่างๆ ในประเทศไทย
>> มีภาพ, มีลิงก์ไปยังเว็บไซต์ภายนอก, และมีแบบฟอร์มสำหรับกรอกข้อความส่งให้เจ้าของเพจได้
>> สามารถเปิดใช้ในอุปกรณ์ 3 ชนิดได้ คือ คอมพิวเตอร์, อุปกรณ์พกพา แนวตั้งหรือแนวนอนได้
>> ให้ นศ.สร้างโฟลเดอร์ และสร้างไฟล์ HTML, CSS หรือไฟล์ประกอบอื่นๆ ไว้ในโฟลเดอร์ดังกล่าว และทดสอบงานจากโฟลเดอร์นี้
>> ให้ นศ.แคปหน้าจอเว็บที่ได้ และจัดทำรายงานประกอบลงไฟล์ word (.docx และ .pdf)

>> สิ่งที่ต้องส่ง
- ไฟล์ zip ของโฟลเดอร์งานทั้งหมด (ให้บีบอัดไฟล์เป็น .zip หรือ .rar)
- ไฟล์รายงาน (.docx และ .pdf) ประกอบด้วย แนวคิด

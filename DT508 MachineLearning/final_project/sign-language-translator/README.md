# โปรแกรมแปลภาษามือ (Sign Language Translator)

เว็บแอปพลิเคชันสำหรับแปลภาษามือเป็นข้อความแบบเรียลไทม์ โดยใช้กล้องเว็บแคมและโมเดล Machine Learning ที่สร้างจาก Teachable Machine โปรเจกต์นี้ถูกสร้างขึ้นเพื่อแปลท่าทางภาษามือของตัวอักษร A-Z และเป็นโปรเจกต์ต้นแบบสำหรับนักศึกษาเพื่อนำไปศึกษาและต่อยอด

## ฟีเจอร์หลัก

*   **การตรวจจับแบบเรียลไทม์:** แปลท่าทางภาษามือ (A-Z) ผ่านเว็บแคมทันที
*   **การสร้างประโยค:** สามารถเพิ่มตัวอักษร, เว้นวรรค, และลบข้อความเพื่อสร้างประโยคที่สมบูรณ์
*   **อ่านออกเสียง (Text-to-Speech):** ฟังเสียงอ่านของประโยคที่แปลได้ (ภาษาอังกฤษ)
*   **แสดงความแม่นยำ:** แสดงค่าความมั่นใจ (Confidence Score) ของตัวอักษรที่ตรวจจับได้แบบ real-time
*   **สถิติการใช้งาน:** นับจำนวนตัวอักษร, จำนวนคำ, และแสดงความแม่นยำเฉลี่ยของการใช้งาน
*   **รองรับคีย์บอร์ด:** ใช้คีย์ลัด (`Enter`, `Spacebar`, `Backspace`) เพื่อความสะดวกในการใช้งาน

## เทคโนโลยีที่ใช้

*   HTML5
*   CSS3 (Bootstrap 5)
*   JavaScript
*   TensorFlow.js
*   Teachable Machine

## แหล่งข้อมูล (Dataset)

โมเดล Machine Learning ในโปรเจกต์นี้ได้รับการฝึกสอน (Train) โดยใช้ข้อมูลจากชุดข้อมูลภาษามือ ASL (American Sign Language) ซึ่งเป็นข้อมูลเปิด (Open-source) จาก Kaggle

*   **Dataset:** [ASL Alphabet on Kaggle](https://www.kaggle.com/datasets/grassknoted/asl-alphabet/data)

## วิธีการติดตั้งและรันโปรเจกต์ (สำหรับนักพัฒนา)

1.  **Clone a Repository:**
    ```bash
    git clone https://[your-repo-url]
    ```

2.  **เตรียมโมเดล Machine Learning:**
    *   โปรเจกต์นี้ต้องการโมเดลที่สร้างและ export มาจาก [Teachable Machine](https://teachablemachine.withgoogle.com/) โดยใช้ Dataset ที่ระบุไว้ข้างต้น
    *   หลังจากดาวน์โหลดโมเดล ให้นำไฟล์ทั้งหมด (`model.json`, `weights.bin`, `metadata.json`) ไปไว้ในโฟลเดอร์ชื่อ `model` ในตำแหน่งเดียวกับไฟล์ `index.html`

3.  **รันบน Local Server:**
    *   เนื่องจากข้อจำกัดด้านความปลอดภัยของเบราว์เซอร์ (CORS policy) คุณ **ไม่สามารถ** เปิดไฟล์ `index.html` โดยตรงจากไฟล์ในเครื่องได้ จะต้องรันผ่าน Local Web Server เท่านั้น
    *   **วิธีที่แนะนำ:**
        *   **ใช้ VS Code Extension:** ติดตั้ง Extension ชื่อ **"Live Server"** แล้วคลิก "Go Live" ที่มุมขวาล่างของ VS Code
        *   **ใช้ Python:** เปิด Terminal หรือ Command Prompt ในโฟลเดอร์โปรเจกต์ แล้วรันคำสั่ง:
          ```bash
          python -m http.server
          ```
          จากนั้นเปิดเบราว์เซอร์ไปที่ `http://localhost:8000`

4.  **เปิดใช้งาน:** เข้าถึงโปรเจกต์ผ่านเบราว์เซอร์ตามที่อยู่ของ Local Server ที่คุณตั้งค่า

## วิธีการใช้งาน (สำหรับผู้ใช้)

1.  เมื่อเปิดหน้าเว็บแล้ว ให้คลิกปุ่ม **"เริ่มต้น"** เพื่ออนุญาตให้เบราว์เซอร์ใช้กล้อง
2.  ทำท่าภาษามือของตัวอักษร (A-Z) ที่ต้องการให้กล้องเห็นชัดเจน
3.  ระบบจะแสดงตัวอักษรที่ตรวจจับได้และค่าความมั่นใจที่กล่อง "ตัวอักษรที่ตรวจพบ"
4.  คลิก **"เพิ่มตัวอักษร"** หรือกดปุ่ม `Enter` บนคีย์บอร์ดเพื่อเพิ่มตัวอักษรนั้นลงในกล่องข้อความ
5.  ใช้ปุ่ม **"เว้นวรรค"**, **"ลบ"**, และ **"ล้าง"** เพื่อแก้ไขข้อความตามต้องการ
6.  คลิก **"อ่านออกเสียง"** เพื่อฟังเสียงอ่านของข้อความที่สร้างขึ้น
